# Windows_Forms_Ekzamen
<h1>1) Создать приложение «Блокнот» (Стандартное Windows приложение «Блокнот» )</h1>
<h3>Реализовать следующий его функционал:</h3>
<h3>Меню:</h3>
<p>Файл: Создать, Открыть, Сохранить, Сохранить как.., Выход</p> 
<p>Правка: Отменить, Вырезать, Копировать, Вставить, Удалить, Выделить все, Время и дата</p>  
<p>Формат: Фон, Шрифт</p>  
<p>Вид: Строка состояния (в ней показывается позиция курсора - строка и столбец)</p> 
<p>Справка: О программе</p> 
<h1>2) Произвести апгрейд приложения «Notepad» до версии «Notepad++», в которой будет добавлен следующий функционал:</h1>
<p>Добавить ToolBar, который дублирует функционал меню</p> 
<p>Добавить функции поиска (найти, найти и заменить и т.д.)</p>
<p>Выбор и смена языка интерфейса</p> 
<p>Выбор цветовой схемы интерфейса</p>   
<p>Выбор синтаксиса языка для открытого документа (C#). При этом выборе, в документе включается синтаксическая подсветка значущих слов, таких как: while, for, if, using и т.д., а также имена переменных;</p>
 <p>Добавить кнопки «Комментировать» и «Раскомментировать» с соответсвующим функционалом;</p> 
 <p>Документы с расширением *.cs открываются сразу с подсветкой;</p>  
 <p>Реализовать Multi Document Interface, в котором, каждый документ открывается в отдельном окне;</p> 
 <p>Окна можно располагать по разному, свернуть все, закрыть все и т.д.</p>
